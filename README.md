# ZeroDrink
html-css 기초 반응형 사이트 제작해보기

<br>

## 링크
https://zero-drink.vercel.app/

<br>

## 기술
- HTML
- CSS


<br>

## 설명
좋아하는 제로 음료수를 소개하는 반응형 사이트입니다.

<br>

## 기능
- 사이트 크기를 조절하면 글씨 크기와 배경 색깔이 바뀝니다.
